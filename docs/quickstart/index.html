Quick Start

#Quick Start

ChronosDB provides a few ways to work with your data:

  • Python package, PyChronos
  • RESTful API
  • web client, when using tshub.io, the SaaS solution.

To learn more about RESTful API, please visit API Reference

#Python

PyChronos is a python package, which provides an interface to the database and tshub.

#Install python package

First, install the package

pip install pychronos

#Create a new time series

Import pychronos, numpy, and pandas.

import pychronos
import pandas as pd
import numpy as np

Login.

username = "johnsmith"
chronos = pychronos.init(username=username, password="mysecret")

Open your default space, which has the same name as your username.

myspace = chronos[username]

Create a new collection.

mycoll_name = "my_collection"
mycoll = myspace.create(myvoll_name)

Create a time series 

kp1 = mycoll.create("kpi1", freq="D", dtype="float")
kp2 = mycoll.create("kpi2", freq="D", dtype="float")
kp3 = mycoll.create("kpi3", freq="D", dtype="float")
kp4 = mycoll.create("kpi4", freq="Q", dtype="int")

You can list time series in your collection.

mycoll.list()

#Save data

Let's create a sample data first.

x1 = pd.Series(np.random.rand(6), index=pd.date_range(start=pd.Timestamp("2000-1-1"), periods=6, freq="D"))
x2 = pd.Series(np.random.rand(6), index=pd.date_range(start=pd.Timestamp("2000-1-3"), periods=6, freq="D"))
x3 = pd.Series(np.random.rand(6), index=pd.date_range(start=pd.Timestamp("2000-1-5"), periods=6, freq="D"))
x4 = pd.Series(np.random.randint(6), index=pd.period_range(start=pd.Period("2000Q1"), periods=6, freq="Q"))

You can save a few series together as long as they are named.

x2.name = "kpi2"
x3.name = "kpi3"
kpi1.save([x2,x3])

Or are in a dataframe with column names corresponding to timeseries names.

df = pd.DataFrame({
  "kpi1": pd.Series(np.random.rand(6), index=pd.date_range(start=pd.Timestamp("2000-1-7"), periods=6, freq="D")),
  "kpi2": pd.Series(np.random.rand(6), index=pd.date_range(start=pd.Timestamp("2000-1-9"), periods=6, freq="D"))
})
kpi1.save(df)

#Retrieve data

You can retrieve data for individual time series using get method in the time series object

kpi1 = mycoll["kpi1"]
kpi1.get()

or directly from collection. mycoll.get("kpi1")

You can retrieve more than one time series as a list of Pandas' Series.

mycoll.get(["kpi1", "kpi2"])

or as a dataframe, as long as all series have the same frequency

mycoll.to_dataframe(["kpi1", "kpi2"])

For more examples, check out the ipython notebook, Examples.ipynb See python notebook with some examples
